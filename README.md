# Transformers Facts API
Creator: Jessica Ganda

This is a mini project about Transformers. It'll display some very fun facts about Transformers which, hopefully, will pull you into this magnificent world of superbeings :D

# Features
Several features listed here:
- get facts about Transformers
- create facts about Transformers
- delete facts based on ID

# API Endpoint
- GET api/facts (get a fact)
- POST api/facts (create a fact)
- DELETE api/facts/:id (delete a fact by ID)
